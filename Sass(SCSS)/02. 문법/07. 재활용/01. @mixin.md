# 01. @mixin

## 📝 재활용 (Mixins)

- 스타일 시트 전체에서 재사용 할 css 선언 그룹을 정의하는 기능
- 약간의 Mixin으로 다양한 스타일을 만들 수 있음
  - `@mixin` : 선언하기 (만들고)
  - `@include` : 포함하기 (사용)

## 📝 @mixin

- `()` : 받아올 내용
- `{}` : 선언하고자 하는 재사용 코드

## 👩‍💻 예제

```scss
@mixin size($w: 100px, $h: 100px) {
  // 기본값 100px로 선언
  width: $w;
  height: $h;
}

.box1 {
  @include size(); // $w, $h 없을 경우 기본값 100px 사용s
}
.box2 {
  @include size(200px, 300px);
}
.box3 {
  @include size($h: 300px); // $w는 기본값, $h는 300px
}
```
